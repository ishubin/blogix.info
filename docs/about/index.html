About

Blogix is a Java-based static website generator. It was inspired by Play framework and Octopress. Blogix allows to create blogs or just simple project websites. Blogix comes out of box with Textille support, syntax highlighting with Highlight.js, tiles-based templates, Freemarker template processor and a simple text file-based database.

Get Started

Getting started with Blogix is easy. Just follow these steps:

  1. Install Download blogix and install it on your local machine
  2. Init your blog Setup your website based on Blogix templates
  3. See how your blog looks Running blogix webserver localy
  4. Publish it Put your blog on github using GitHub Pages or any other hosting
